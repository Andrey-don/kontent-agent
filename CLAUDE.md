# CLAUDE.md — инструкции для работы с проектом

## Что это за проект

Контент-агент для вайбкодера. Система файлов и промптов для производства контента в Telegram-канал и ВКонтакте.

## Ключевые файлы

| Файл | Назначение |
|------|-----------|
| `brief.md` | Кто такой автор, на кого пишет, зачем |
| `audience.md` | Исследование аудитории: боли, вопросы, конкурентный контент |
| `tone-of-voice.md` | Голос автора: правила, примеры, словарь — **прикреплять к каждому запросу на пост** |
| `content-plan.md` | 20 тем на месяц с типами и крючками |
| `posts-draft.md` | Готовые черновики постов |
| `PLAN.md` | Дорожная карта проекта |

## Обязательные правила при написании постов

1. **Всегда читать `tone-of-voice.md`** перед написанием — голос важнее структуры
2. **Результат первым** — пост начинается с итога, не с раскачки
3. **Пост про клиента**, не про инструмент
4. **Кодовое слово** в каждом CTA (одно слово — один пост)
5. **Провокационный вопрос** перед призывом к действию
6. Длина поста: 150–300 слов
7. Эмодзи: 3–5 штук, только там где усиливают смысл

## Аудитория

- Владельцы малого бизнеса (кафе, салоны, мастера, магазины)
- Эксперты-фрилансеры
- Стартаперы

## Площадки

- ВКонтакте (адаптация)
